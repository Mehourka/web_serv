About the project

The goal for this project was to create a webserver from scratch in C++, managing HTTP requests and sending the correct responses in return to the clients. To achieve this, we looked at the RFC to learn about the specifics of the HTTP protocol and also based ourselves on Nginx to create a configuration file for our webserver. We also implemented a CGI to run external scripts, most notably for uploading files to the server through a html form.
